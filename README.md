# clj-snake
Functional approch to Snake Game in Clojure powered by [rx-clojure](https://github.com/Vikasg7/rx-clojure).  

# Usage
* Install [Leiningen](https://leiningen.org/)  

* ````console
  git clone https://github.com/Vikasg7/clj-snake.git
  cd clj-snake
  lein trampoline run
  ````
